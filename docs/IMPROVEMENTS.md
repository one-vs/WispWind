# План улучшений

Идеи и техдолг, отсортированы по соотношению польза/усилия. Статусы: `[ ]` не начато, `[~]` частично, `[x]` сделано.

## Высокий приоритет

- [ ] **Realtime-режим: привести к уровню batch.** Realtime-путь (STT_MODE=realtime) не получил
  последних улучшений: нет сохранения аудио, watchdog есть только на финальный результат,
  live-вставка через посимвольный backspace хрупкая в некоторых полях. Либо довести, либо
  скрыть из настроек до доработки.
- [ ] **Автовосстановление после сбоя STT из recordings/.** Сейчас WAV сохраняется, но повторная
  отправка — ручная. Добавить в дашборд кнопку «Retranscribe» напротив записи и/или
  автоматический ретрай отложенных файлов при старте.
- [ ] **Автоуровень микрофона.** inputGain захардкожен (3.0). Сделать AGC: замерять RMS первых
  секунд и подстраивать усиление, либо вынести gain в настройки.

## Средний приоритет

- [ ] **Поиск по истории** в дашборде (полнотекстовый по user_history, SQLite FTS5).
- [ ] **Экспорт истории** (markdown/json за период) из дашборда.
- [ ] **Пин панели истории**: кнопка-скрепка, чтобы панель не закрывалась при потере фокуса;
  drag за заголовок.
- [ ] **Клик по трею**: левый клик по иконке — открыть панель истории (сейчас только хоткей).
- [ ] **Словарь подсказок STT**: пользовательский список терминов (имена, бренды, аббревиатуры),
  автоматически подставляемый в STT_PROMPT.
- [ ] **Мультипрофили промптов LLM**: быстрое переключение «диктовка/письмо/код-комментарий»
  из трея или отдельными хоткеями.
- [ ] **Уведомление об ошибке STT**: сейчас ошибка видна только в логе — показать краткий
  статус в виджете (красная волна/крестик) перед скрытием.

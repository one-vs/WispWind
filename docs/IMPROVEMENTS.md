# План улучшений

Идеи и техдолг, отсортированы по соотношению польза/усилия. Статусы: `[ ]` не начато, `[~]` частично, `[x]` сделано.

## Высокий приоритет

- [x] **Realtime-режим объявлен legacy** (решение 2026-07-15): batch с gpt-4o-mini-transcribe
  работает лучше. Опция скрыта из дашборда; код остаётся, но не развивается. Кандидат на
  полное удаление.
- [x] **Ретранскрибация из recordings/.** На странице History дашборда — таблица WAV-бэкапов
  с аудиоплеером и кнопкой Transcribe. Отправка только ручная (никакого автозапуска, чтобы
  не генерировать неожиданные расходы); результат попадает в историю и usage.
- [x] **Автоуровень микрофона (AGC).** MIC_GAIN=auto по умолчанию: усиление адаптируется к
  целевому RMS 0.15 (быстрое снижение при клиппинге, медленный подъём). Числовое значение
  фиксирует gain как раньше.

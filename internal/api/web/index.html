Help & Setup

How to configure WispWind, use hotkeys, and troubleshoot issues.

Quick Start

On first launch the app creates a local SQLite database and reads settings from the .env file next to the executable. If .env is missing, the app still runs but you must set your API key through this dashboard (Settings page) before dictation will work.

  1. Copy .env.example to .env in the same folder as wispwind.exe.
  2. Add at least one API key: OPENAI_API_KEY=sk-... or DEEPGRAM_API_KEY=...
  3. Restart the app so the key is picked up at startup.
  4. After the first run the admin panel is the source of truth for all settings.

Hotkeys

Global hotkeys work from any application. Default mode is hold. You can switch to toggle in Settings.

Action Hold mode Toggle mode
Start recording Hold Ctrl + Space Press Ctrl + Space
Stop / Send Release the keys Press Ctrl + Shift + Space
Cancel Press Esc while recording to discard audio and abort without sending anything to the API.

Hotkey combinations can be changed in the Settings page, but a restart is required for the change to take effect.

How It Works

  1. Record — press and hold the start hotkey. A green waveform widget appears near your cursor.
  2. Speak — your voice is captured as PCM audio at 24 kHz.
  3. Release — audio is sent to the configured STT provider (OpenAI or Deepgram).
  4. LLM polish — if enabled, the raw transcript is passed through an LLM for formatting / cleanup.
  5. Paste — the final text is typed into the window that was active when you started recording.

In realtime mode the text streams live as you speak. In batch mode nothing is sent until you release the hotkey.

Troubleshooting & Logs

If dictation stops working or the widget does not appear:

  • Check that your API key is set and has available quota.
  • Verify the app is not already running — only one instance is allowed.
  • Open the Logs page in this dashboard to see runtime errors.
  • Log files are stored locally in the app data folder and rotate daily.

Common error: "invalid_api_key" — the key was not found or is expired. Add it via Settings and restart.
